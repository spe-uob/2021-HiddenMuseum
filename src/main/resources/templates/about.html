About us

This project builds a prototype “advanced search” tool that enables staff, students, public and researchers to browse and filter and search database records to find specific objects in the collection. The project builds a search mechanism on top of the OpenData tool which has an API.

The main search bar has “fuzzy search”. This means it is not case sensitive and it searches all of the fields in the chosen dataset. The filters work by having a dropdown menu to select the field. A dropdown to choose whether to refine or exclude the search. A text box to pattern match directly against the value in the selected field. This is case sensitive and must match exactly in accordance with the corresponding data set found on opendata.bristol.gov.uk

There is an infographics page that displays the number of artworks per year and displays the artworks for each year when the year is clicked.

Bristol Museum is working in conjunction with students from the University of Bristol to digitise its 2 million objects held on display and also in the museum stores, covering Archaeology, Natural Sciences, Art and History collections. Data generated about the objects is a useful tool for researchers across disciplines, but the data is not currently accessible apart from on an outdated website musuems.bristol.gov.uk.
